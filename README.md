# calcisphere

A responsive calculator web application built using HTML, CSS, and JavaScript. The project is designed with a clean user interface and supports basic arithmetic operations through interactive frontend functionality.

## Features
- Basic arithmetic operations
- Interactive calculator interface
- Responsive design
- Clean and modern UI
- Smooth button interactions

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
- `index.html`
- `style.css`
- `script.js`
